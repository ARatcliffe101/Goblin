GOBLIN Help (v5)

GOBLIN — Graphical Output Builder for Language Integration & Narration

This tool helps you author SSML by highlighting text and applying controlled SSML elements. The SSML output is always visible and portable.

Pronunciation Manager

The Pronunciation Manager is a structured, reusable library for common e-learning pronunciations and interpretations. Each entry includes:

  • Term (the written text)
  • Method (phoneme, sub, or say-as)
  • Parameters (e.g., IPA string, alias text, interpret-as)
  • Category/Tags (optional)
  • Notes (optional)

Actions:

  • Apply to selection: wraps currently highlighted text with the chosen method.
  • Insert at cursor: inserts the term already wrapped with its SSML.
  • Apply to all occurrences: best-effort helper that wraps all exact text matches found in the editor’s text nodes.
  • Import/Export: move your pronunciation library between machines using JSON.

Because “apply to all” is textual matching, review the results to ensure no unintended matches were wrapped.

Tag Inspector

Click any highlighted SSML region (or inserted marker) in the editor to open the Tag Inspector. In v5, phoneme/sub/say-as tags can be sent directly to the Pronunciation Manager.

Undo / Redo

Use Undo and Redo to step through editor history.

Portability notes

  • The output is standards-based SSML 1.1, with a single <speak> root.
  • Engines differ in which tags/attributes they support; always validate in the target platform.

References
